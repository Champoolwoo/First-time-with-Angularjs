# เริ่มต้นการใช้งาน Angularjs

###สร้าง Folder
```bash
project{
  public{
    css
    js
    page
    pic
    index.html
  }
  server.js
  package.json
}

```
###จากนั้นใช้คำสั่งด้านล่างนี้ แล้วจะได้ไฟล์ package.json
```bash
$ npm init
```
###จากนั้นใช้คำสั่งด้านล่างนี้ แล้วจะได้ Folder node_modules
```bash
$ npm install nodemon
```
###ทำการติดตั้ง Espress
```bash
$ npm install express
```
###คำสั่งในการ Run
```bash
$ nodemon server
```
